# skedge

Schedule Appointments, send reminders, gain repeat business, and increase your five star reviews!

# Installation

To install the required modules, run the command

```
$ npm i
```

---

To start up your local dev envirement run:

```
$ npm run dev
```

Run Test

```
$ npm t
```

Serve up docs

```
$ npm run dev
```
